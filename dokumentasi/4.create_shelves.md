## Create Shelves

Langkah pertama, pada dashboard, silahkan pilih menu Shelves/Rak

[[gambar/setup_booksack.png]]

Kemudian, silahkan anda pilih new shelf

[[gambar/add_shelves.png]]

Setelah itu, silahkan isi data sesuai dengan keinginan anda

[[gambar/data_shelf.png]]

Setelah shelves tebuat, silahkan pilih create new book
[[gambar/create_book.png]]

Silahkan isi data sesuai dengan keinginan anda

[[gambar/new_book.png]]

Setelah buku terbuat, selanjutnya silahkan anda add a chapter

[[gambar/add_chapter.png]]

Silahkan sesuaikan isi chapter dengan keinginan anda

[[gambar/new%20chapter.png]]

Silahkan klik create a new page

[[gambar/new%20page.png]]

Kemudian, pada halaman page silahkan isi sesuai kebutuhan anda

[[gambar/page.png]]

Dan berikut adalah contoh dalam buku Visi Misi Perusahaan

[[gambar/halaman%20employee%20handbook.png]]
